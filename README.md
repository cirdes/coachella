Coachella
=========

Eventick Auto Checkin


### Set up & Install dependencies

```
$ git clone git@github.com:cirdes/coachella.git
$ cd coachella
$ bundle install
```

### Server

```
$ middleman
```

### Dist

```
$ middleman build

```
